# Java Compiler

Compiler Implementation in Java.

Each package/folder have its own README file.

## Compile and Test

**Compile** this code with following commands:

``` bash
javac lexer/*.java
```

``` bash
javac parser/*.java
```

``` bash
javac inter/*.java
```

``` bash
javac symbols/*.java
```

``` bash
javac main/*.java
```

To **test** the code:

``` bash
java main.Main __inputfile__ __outputfile__
```
